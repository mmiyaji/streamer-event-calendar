プライバシーポリシー

制定日・最終更新日：2026年7月23日

本ページは、個人が無償で運営する「Streamer Event Calendar」（以下「本サービス」）における情報の取扱いを説明するものです。

1. 運営者が直接収集する情報

本サービスには、氏名、メールアドレス、住所、電話番号、アカウント情報等を入力するフォームはなく、運営者は本サイト上で利用者の個人情報を直接収集しません。

2. Cookieおよびアクセス解析

本サービス独自のCookie、広告、トラッキング機能、アクセス解析サービスは使用していません。今後これらを導入する場合は、本ポリシーを更新します。

3. GitHub Pages等の外部サービス

本サービスはGitHub Pagesおよび外部CDNを利用しています。これらの事業者は、サービス提供、セキュリティ、不正利用防止等の目的で、IPアドレス、利用日時、端末・ブラウザ情報、アクセスログ等を取得または保存する可能性があります。取扱いは各事業者の規約・ポリシーに従います。

4. カレンダー購読

ICSまたはwebcalリンクを利用した購読処理は、利用者のOS、カレンダーアプリ、クラウドカレンダー事業者等によって行われます。購読情報や利用状況の取扱いは、各サービスの規約・プライバシーポリシーに従います。

5. 外部リンク

本サービスから移動した外部サイトで行われる情報収集について、運営者は管理しません。リンク先のプライバシーポリシーをご確認ください。

6. GitHub Issuesでの問い合わせ

GitHub Issuesを利用して問い合わせる場合、投稿内容、GitHubアカウント名、プロフィール等が公開される可能性があります。個人情報や非公開情報を投稿しないでください。

7. 生成AIの利用

本サービスではイベント情報の収集・整理等に生成AIを利用する場合がありますが、本サイトの閲覧者に関する個人情報を生成AIへ入力することを目的とした仕組みはありません。

8. ポリシーの変更

運営方法、利用サービス、法令等の変更に応じて、本ポリシーを予告なく変更することがあります。変更後の内容は本ページに掲載した時点から適用されます。

9. お問い合わせ

本ポリシーに関する問い合わせは、GitHub Issuesをご利用ください。公開投稿になる点にご注意ください。
